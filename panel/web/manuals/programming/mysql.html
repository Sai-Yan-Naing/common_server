MySQLデータベースへの接続例

まだMySQLのデータベースを作成していない場合は、 MySQLのデータベースを作成してください。

  1. phpの場合（PDO）
    ※下記の例では、MySQLに接続して、SELECT文を実行しております。

    $dsn = 'mysql:host=*****.winserver.ne.jp;dbname=example_db';

    $user = 'example';
    $password = 'example';

    $pdo = new PDO($dsn, $user, $password);
    $stmt = $pdo->query("SELECT * FROM example_table);

    $pdo = NULL;

  2. Pythonの場合
    ※下記の例では、MySQLに接続して、SELECT文を実行しております。

    import MySQLdb

    connector = MySQLdb.connect(db="example_db", host="MySQLサーバーのIPアドレス", user="example", passwd="example")
    cursor = connector.cursor()
    cursor.execute("select * from example_table;")

    result = cursor.fetchall()

    cursor.close()
    connector.close()

  3. ASP.NETの場合
    共用サーバー用の「MySql.Data.dll」が必要になります。 ご利用をご希望の場合には、お気軽にご連絡ください。
